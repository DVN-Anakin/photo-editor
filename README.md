# photo-editor
JavaScript online photo editor
https://dvn-anakin.github.io/photo-editor/

## Functionalities
- RGB to RBG
- RGB to BGR
- RGB to GBR
- RGB to GRB
- Greyscale
- Sepia effect
- Invert colors
